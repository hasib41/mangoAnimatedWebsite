# Sound effects (served at /sounds/…)

Drop the lamp switch clicks here:

- `lamp-on.mp3`  — plays when the lamp turns ON
- `lamp-off.mp3` — plays when the lamp turns OFF

Keep them short (< ~300 ms) and small. If you only have one click,
save it as both filenames (the code falls back to lamp-on for off).

Use royalty-free / CC0 clips so YouTube Shorts don't get Content ID claims.
