# MONOLITH — Gemini Image Prompts

Build target: a single full-screen **3D draggable coverflow** — six vertical poster cards arranged on a curved arc; the centred card scales forward and glows in its accent colour while neighbours rotate away in 3D. Drag, scroll, arrows, or autoplay glide through them.

**6 cards.** Each is a **surreal, monumental dream-building** — "impossible architecture" shot like a cinematic film poster. Each scene is dominated by ONE signature colour (it drives that card's on-screen accent), set against a clean, calm sky so the structure reads instantly even at thumbnail size.

**Aspect ratio:** vertical **3:4** (target ~**1200 × 1600**). The monument is centred and vertical so it reads both as a small tilted side-card and as the large hero card.
**Background removal:** NONE — these are full-bleed card backgrounds. Save as **JPG**.
**Save each Gemini output in `~/Downloads` with the exact filename below**, then drop into `public/photos/monolith/` (overwrite the placeholders already there).

**House style (apply to every prompt):** photoreal architectural render / fine-art photograph, golden-hour or soft overcast daylight, vast empty scale, minimalist, strong single-point or symmetrical composition, gentle atmospheric haze, subtle film grain, shot on a wide architectural lens. **Forbidden in every image:** people, faces, cars, text, signage, watermarks, logos, clutter. Clear simple sky — **never pure white**.

## Processing pipeline (when files land in ~/Downloads)
- Detect by the filenames above. **No background removal** (full-bleed scenic backgrounds).
- Re-encode each as quality-85 JPG, resize so the long edge ≤ **1600px** (3:4 → ~**1200 × 1600**).
- Overwrite the placeholders in `public/photos/monolith/`. The `<img>` tags already carry `width="1200" height="1600"` — adjust only if you export a different size.
- Accent colours are read from each card's `data-accent` in `monolith/index.html`; if you change a scene's dominant colour, update its `data-accent` to match (it drives the glow, caption colour, and ticks).
- Optional: drop a real `public/favicon-monolith.png` (square logo mark) to replace the seeded copy.
